# HoloSettings (app)
These are experimental apps and not stable. WORK IN PROGRESS
# App file meaning:
1. "_src": The app with Target API 18. Updates are coming out later (typically 1 day after API 24 app's update)
2. "Intent" suffix: A specific sub-menu got redirected to an intent (external fragments of native Settings app)
3. "Fix" suffix: A specific sub-menu got fixed, without the usage of an intent
